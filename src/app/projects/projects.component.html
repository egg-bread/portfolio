A skincare journal app where users create daily logs rating their skin condition, what products they use in their skin routines (AM and PM routine), and attach pictures to track their skin condition. Built in React Native with a Java backend as a Spring Boot app. The backend API uses MySQL and Spring Data JPA. This is still a project in the works with other features planned.
